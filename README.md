# Tutorial repository
This is the official repository for the Kvantify Qrunch tutorials. Each folder contains one tutorial and the files required by the tutorial. For each tutorial it is required to have a working conda environment with `qrunch` installed. To find out how to install `qrunch`, please see the [installation instructions](https://xkcd.com/404/). It is further recommended to use the tutorials in combination with an IDE with working code-completion, please see [our setup instructions](https://xkcd.com/404/) for various IDEs.

The tutorials are given as jupyter notebooks and should be self-explanatory. If something does not work according to expectations, please [write us an email](mailto:support-qrunch@kvantify.dk?subject=Tutorial%20defect&body=Hi,%0AThe%20tutorial%20TUTORIAL_NAME%20does%20not%20work%according%20to%20my%20expectations.%20I%20have%20observed%20...%20).
